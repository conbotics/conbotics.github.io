    MMSegmentation Installation Guide for NVIDIA Jetson

    This guide provides step-by-step instructions for installing the MMLabSuite packages on an NVIDIA Jetson device. These instructions will help you set up the PyTorch ecosystem with specific versions that are compatible with Jetson hardware.

    Prerequisites

    • NVIDIA Jetson device with JetPack 5.1 or later
    • Internet connection
    • Git installed

    Installation Steps

    1. Create Working Directory and Python Virtual Environment

    First, we'll create a dedicated directory and virtual environment for the MMLabSuite:

    # Create a Software directory
mkdir -p Software
cd Software

# Create virtual environment named 'mmlab' using Python's built-in venv
python3 -m venv mmlab

# Activate the virtual environment
source mmlab/bin/activate

    2. Install PyTorch for Jetson

    Install the specific PyTorch version for Jetson:

    # Download PyTorch wheel for Jetson
wget https://developer.download.nvidia.com/compute/redist/jp/v51/pytorch/torch-1.14.0a0+44dac51c.nv23.02-cp38-cp38-linux_aarch64.whl

# Install the downloaded PyTorch wheel
pip install torch-1.14.0a0+44dac51c.nv23.02-cp38-cp38-linux_aarch64.whl

    3. Install TorchVision

    # Install torchvision
pip install torchvision==0.15.0

    5. Clone and Install mmcv

    # Clone mmcv repository
git clone https://github.com/open-mmlab/mmcv.git
cd mmcv

# Install the specific version
git checkout v2.0.0rc4

# Install the package
pip install -v -e .

# Return to the Software directory
cd ..

    6. Clone and Install mmengine

    # Clone mmengine repository
git clone https://github.com/open-mmlab/mmengine.git
cd mmengine

# Install the specific version
git checkout v0.10.4

# Install the package
pip install -v -e .

# Return to the Software directory
cd ..

    7. Clone and Install mmsegmentation

    # Clone mmsegmentation repository
git clone https://github.com/open-mmlab/mmsegmentation.git
cd mmsegmentation

# Install the specific version
git checkout v1.2.2

# Install the package
pip install -v -e .

# Return to the Software directory
cd ..

    8. Install mmdet

    # Install mmdet
pip install mmdet==3.3.0

    9. Install ftfy

    # Install ftfy
pip install ftfy==6.2.3

    10. Fix PyTorch Alpha Version Compatibility Issue

    Since we're using an alpha version of PyTorch, we need to modify the mmengine/dist/dist.py file to ensure compatibility:

    # Navigate to the mmengine directory
cd mmengine

# Create a backup of the original dist.py file
cp mmengine/dist/dist.py mmengine/dist/dist.py.backup


# With nano:
nano mmengine/dist/dist.py

    Find and comment out the following section in the file:

    # Comment out the following section:
# def get_reduce_op(name: str) -> torch_dist.ReduceOp:
#     op_mappings = {
#         'sum': torch_dist.ReduceOp.SUM,
#         'product': torch_dist.ReduceOp.PRODUCT,
#         'min': torch_dist.ReduceOp.MIN,
#         'max': torch_dist.ReduceOp.MAX,
#         'band': torch_dist.ReduceOp.BAND,
#         'bor': torch_dist.ReduceOp.BOR,
#         'bxor': torch_dist.ReduceOp.BXOR,
#     }
#     if name.lower() not in op_mappings:
#         raise ValueError(
#             f'reduce op should be one of {op_mappings.keys()}, bug got {name}')
#     return op_mappings[name.lower()]

    Save and exit the editor (for nano: CTRL+O, Enter, CTRL+X).

    11. Verify Installation

    After completing all the steps, you can verify your installation by importing the packages:

    # Activate the virtual environment if not already activated
source Software/mmlab/bin/activate

# Start Python
python

# Try importing the packages
>>> import torch
>>> import torchvision
>>> import mmcv
>>> import mmengine
>>> import mmdet
>>> import mmsegmentation
>>> print(f"PyTorch version: {torch.__version__}")
>>> print(f"MMCV version: {mmcv.__version__}")
>>> print(f"MMEngine version: {mmengine.__version__}")
>>> print(f"MMDet version: {mmdet.__version__}")
>>> exit()
