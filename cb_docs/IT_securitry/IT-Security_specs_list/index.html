    Sicherheitsrelevante Aspekte der App: - Läuft auf Android Tablets - Stack: - Capacitor JS (native App mit weblayer) - React + TS + Vite - relevante dependencies: - roslibjs für (unverschlüsselte) Kommunikation mit ROS bridge - "@capacitor/filesystem" für Zugriff (read/write) auf Dateien (z.B. Bilder und Raumplandateien)
    - Live Updates für Tablets per Plugin "@capawesome/capacitor-live-update" - Erlauben Updates innerhalb der App ohne App-Store - Sources für Weblayer (html, css, js) werden als Bundle per Cloud zur Verfügung gestellt - Capawesome Cloud registriert Tablets, die die App nutzen und tracked welches Gerät auf welcher Version läuft
      - Updates müssen durch Nutzer aktiv ausgewählt werden - Bundles werden mit private key signiert, sodass App keine "fremden" Updates akzeptiert - Self-hosting des Bundle-Servers ist eine Möglichkeit - Tablet muss für Updates mit Internet verbunden sein, sonst nur mit dem Roboter
