    Software Team Guidlines

    We use several guidlines and style ideas to make the software development in our team as easy as possible. You can go through the following sections to see what we set as our rules / guidlines and where yo can extend your knowledge. We also talk regularly about these in our Software Readin Club every second Friday.

    Code Style

    We have our own set of Code Style guidlines. Some code might not follow these rules since we established them recently. However, if you write new code or have to make changes to existing code, try to follow them as much as possible!

    More rules can be found here. But it is a lot to read and our own rules always applie before any of these: Clean Code Cheat Sheet
